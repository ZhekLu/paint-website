Paint Website

A free professional web tool to realize your creative ideas.
Created by EUGENE ROTKOVICH

About

Site designed with an open source HTML (Bootstrap3) theme created by Code Design. This project was created to practice a Python and JavaScript languages, Django framework and OOP programming.

Paint

Website contains professional paint tool. There you can draw figures like rectangle, ellipse, polygon, undo last drawing, change colors and save or upload pictures.

Gallery and Profile

This project contains a gallery for viewing your and other people creations, post them for all and comment on interesting pictures. You can use this project without registrations, but only authorized users can see their own gallery and upload new pictures.

Technologies

In project were used Python, JavaScript, HTML, CSS
Django Framework
